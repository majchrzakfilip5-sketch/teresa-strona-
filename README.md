# Teresa - strona

Projekt zawiera **3 wersje** strony.

## Najszybciej: odpal wszystkie 3 naraz

Kliknij dwukrotnie plik:

```
START-WSZYSTKIE-3-WERSJE.bat
```

Wersje 1 i 2 otworzą się od razu w przeglądarce. Wersja React wystartuje w osobnym oknie.

## Wymagania

- **Wersja Chris** i **Wersja 1** — nic nie trzeba, to zwykły HTML (działa dwuklikiem).
- **Wersja React/Vite** — wymaga **Node.js** (pobierz z https://nodejs.org, wersja LTS).
  Pierwsze uruchomienie samo doinstaluje resztę, to potrwa chwilę.

## Każda wersja osobno

| Wersja | Jak odpalić | Folder |
|--------|-------------|--------|
| Chris (HTML) | dwuklik `START.bat` lub `index.html` | `stronateresaChris/` |
| Wersja 1 (HTML) | dwuklik `START.bat` lub `index.html` | `theresa-app-landing/teresawersja1/` |
| React/Vite | dwuklik `START.bat` | `theresa-app-landing/` |

### Wersja React ręcznie (jak wolisz z konsoli)

```
cd theresa-app-landing
npm install
npm run dev
```
