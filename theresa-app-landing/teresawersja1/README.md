# Teresa App Landing Page

Responsywny landing page aplikacji Teresa, AI-przewodnika po telefonie dla osób starszych.

## Uruchomienie

Projekt nie wymaga instalowania zależności. Otwórz `index.html` w przeglądarce albo uruchom prosty serwer lokalny:

```bash
python3 -m http.server 8000
```

Następnie wejdź na `http://localhost:8000`.

## Struktura

- `index.html` - treść i struktura strony
- `styles.css` - stylistyka oraz responsywność
- `script.js` - menu mobilne, animacje i formularz demonstracyjny
- `assets/` - obrazy używane na stronie

## Współpraca

1. Sklonuj repozytorium.
2. Utwórz osobny branch, np. `feature/nazwa-zmiany`.
3. Wprowadź i sprawdź zmiany lokalnie.
4. Wyślij branch i otwórz Pull Request do `main`.

## Status formularza

Formularz listy oczekujących działa obecnie demonstracyjnie po stronie przeglądarki. Przed publikacją produkcyjną należy podłączyć go do wybranej usługi lub API.
